✅ CIA 三元素
是資訊安全最重要的核心概念。

🔐 CIA 是什麼？
英文縮寫	全名				中文意思	重點是？
C		Confidentiality	機密性	防止資料被不該知道的人看到
I		Integrity		完整性	確保資料沒被亂改或損壞
A		Availability	可用性	確保需要時，資料能正常使用

🧠 舉例幫助理解：
✅ 機密性 C	資料只能某些人看，例如：你的成績單不能被同學看到
✅ 完整性 I	成績不能被偷偷改掉；網銀轉帳金額不能被駭客改成 100 萬
✅ 可用性 A	當你要登入網銀或查資料時，系統不能掛掉；要隨時能用

🔐 三者缺一不可
如果少了其中一項，就會發生問題：
少了什麼？	結果
❌ 機密性	資料被偷看或外洩（例如帳號密碼被駭）
❌ 完整性	資料被改（例如成績被篡改）
❌ 可用性	系統掛掉不能用（例如網站被 DDoS 攻擊）

🧱 延伸補充：常見的資安攻擊如何破壞 CIA？
攻擊手法			破壞哪一項？
資料外洩			機密性（C）
網頁被改內容		完整性（I）
DDoS 網站打不開	可用性（A）

Rootkit 是一種惡意軟體（Malware），它的主要特點是：
🎯 隱藏存在、控制系統、繞過偵測
✅ 定義：
Rootkit 是一組工具或程式，可以讓攻擊者取得 系統的最高權限（Root 或 Admin），並且 隱藏自己的存在，使系統管理員或防毒軟體難以察覺。
🧠 命名由來：
「Root」：在 Linux/Unix 系統中代表最高權限的使用者。
「Kit」：指的是工具組，成套工具。
合起來就是「讓駭客取得 Root 權限並且躲起來的工具組」。

**惡意軟體（Malware）**是指惡意軟體程式，這是一個非常廣泛的術語，用來描述那些在我們的電腦或網路上的其他裝置中，執行有害功能的軟體。

這些惡意軟體可能會：
蒐集按鍵記錄（Keystrokes），然後把這些資料傳送給駭客。
使你的電腦成為**殭屍網路（Botnet）**的一部分，由電腦內安裝的惡意軟體來控制這些行為。
當你上網時，不斷彈出訊息與廣告，這些彈出式廣告就是由惡意軟體造成的。
或者是傳統的病毒（Virus）或蠕蟲（Worm），導致電腦出問題，如：
=>加密你的資料
=>刪除硬碟中的檔案
=>讓你在移除惡意軟體後，還得試圖恢復剩下的資料

🔹【惡意軟體種類與特性整理】
名稱								特性說明
病毒（Virus）						必須依賴人為行為（例如打開檔案）才能散播，會附著在其他程式中。
加密惡意軟體（Crypto-malware）		加密你電腦上的所有資料，常與**勒索軟體（Ransomware）**結合出現，藉此要求贖金。
蠕蟲（Worm）						不需要人為操作就可自我傳播，會自動從一台電腦擴散到另一台。
特洛伊木馬（Trojan Horse）			看起來像是無害程式，但實際上內含惡意程式碼，一旦安裝即執行破壞行為。
Rootkit（根套件）					深層隱藏於系統中，讓駭客得以潛伏，通常難以偵測與移除。
按鍵記錄器（Keylogger）			紀錄使用者的鍵盤輸入行為（如帳號密碼），並傳送給攻擊者。
廣告軟體（Adware）				不斷跳出廣告，干擾使用者，可能包含追蹤功能。
間諜軟體（Spyware）				悄悄監控你的行為與資料，例如瀏覽記錄、密碼、信用卡資訊。
殭屍網路（Botnet）				攻擊者控制多台被感染的電腦（Bot），用於進行大規模攻擊如DDoS。

🔹【惡意軟體入侵流程與手法】
尋找漏洞（Vulnerability）：作業系統或應用程式的漏洞
使用者點擊了有問題的電子郵件連結或看似正常的網站內容，執行惡意程式：
一旦執行，惡意軟體會被安裝，可能會包含後門程式（Backdoor），讓駭客可繞過防火牆再次進入系統。

殭屍網路安裝：攻擊者可安裝更多軟體並控制系統行為。
散佈途徑：電子郵件連結
惡意網站（Drive-by download）：只要造訪網站就會自動下載檔案
蠕蟲：可不需使用者操作就感染其他電腦

🔹【如何防範惡意軟體】
保持作業系統與應用程式更新
即時安裝安全修補程式（Patch）
不要點擊不明郵件內的連結
直接在瀏覽器輸入網址，不要透過來路不明的跳轉
注意 Adobe Flash、PDF 等常被攻擊的應用程式漏洞

📘電腦病毒與蠕蟲介紹
電腦病毒（computer viruses）人與人之間傳播病毒借來的。
電腦病毒可以在裝置之間自行傳播，而且不一定需要你去點擊什麼東西，但它需要在你的電腦上執行某個程式才能啟動其複製的功能。
病毒可以透過檔案系統或網路來複製傳播。有時候只要電腦中存在可執行檔案（executable），它就能找到方式在你網路中的其他裝置中散播。

執行中的病毒不一定會造成災難性的結果，有些可能只是執行一些不影響其他功能的簡單操作。
有些病毒是隱形的，幾乎無法察覺其存在，而有些則會不斷跳出煩人的彈出視窗，非常明顯。
這就是為什麼我們需要安裝防毒軟體。防毒軟體會檢查系統中是否有已知病毒正在執行，
一旦發現可疑程式試圖執行，它會立即阻止並隔離它。
由於每週都有數千種新病毒被發現，我們也要不斷更新病毒定義（signature list），讓防毒軟體能識別並阻止新病毒。

病毒種類非常多，其中最常見的包括：
附著在應用程式中的病毒：當你執行程式時，病毒也會一併被執行。
開機磁區病毒（boot sector virus）：它不依賴作業系統，而是直接駐留在開機磁區，要移除它就必須對磁區進行處理。
腳本病毒（script virus）：可存在於作業系統或瀏覽器中。
巨集病毒（macro virus）：通常藏在試算表或文書處理軟體的巨集中。

其中一種破壞力特別大的病毒是 蠕蟲（worm）。
🐛蠕蟲病毒（Worm）的特點與範例：WannaCry
蠕蟲是一種能自動在系統間移動的病毒，不需要使用者做任何事就能利用網路快速傳播。
由於蠕蟲傳播非常快，一旦開始感染，很難控制。若某個漏洞影響很多系統，蠕蟲很快就能全面感染。若這種蠕蟲已被辨識，
防火牆或入侵防禦系統（IPS）可以利用簽章檔過濾掉這類惡意流量。

WannaCry（又稱 WannaCrypt）傳播流程：
一台已感染的電腦正在執行 WannaCry 蠕蟲。
該電腦會在網路中尋找其他存在漏洞的設備。
WannaCry 利用了 Microsoft 的 SMB v1 漏洞（Server Message Block），這是 Windows 用來傳輸檔案的協定，因此影響範圍廣大。
找到漏洞裝置後，它會利用EternalBlue（一個外部工具）來攻擊該裝置。
EternalBlue 會安裝一個後門程式（backdoor），此程式接著會聯絡控制中心下載 WannaCry。
下載後 WannaCry 開始執行並加密該裝置的資料。
被感染的裝置再繼續尋找下一個漏洞裝置，整個流程持續循環。
WannaCry 可以不需任何使用者操作，就橫越全世界進行傳播與加密攻擊。

✅重點整理：
項目	說明
🔍 電腦病毒	像人類病毒一樣可自我複製傳播，需執行程式才會生效。
📂 傳播方式	可透過檔案系統、網路、自動執行傳播。
🙈 表現形式	有的病毒隱藏不易察覺，有的會彈出視窗等明顯異常。
🛡️ 防護手段	使用防毒軟體、維持病毒定義檔最新狀態。
🧬 病毒種類	應用程式型、開機磁區型、腳本型、巨集型。
🐛 蠕蟲（Worm）	可自動透過網路傳播，不需使用者互動。
💣 WannaCry	著名蠕蟲病毒，利用 SMBv1 漏洞 + EternalBlue 傳播並加密資料。

🔍 third party 是甚麼？
指不是你自己或主要開發商的外部公司或組織。
舉例：你用的軟體是 A 公司做的，但它可能會使用 B 公司（第三方）做的工具或程式碼，像是殺毒軟體中的「第三方工具」。

🔍 eternal blue 是甚麼？
EternalBlue 是一個微軟 Windows 系統中的漏洞攻擊工具，黑客用它來利用 Windows 的一個安全漏洞（SMB v1協定的漏洞），以便入侵電腦並安裝惡意軟體。
它就是 WannaCry 蠕蟲用來入侵其他電腦的「鑰匙」。

🔍 巨集病毒 運作原理？
巨集是文書或試算表軟體（像 Word、Excel）內建的一種自動化指令程式，可以幫你自動完成工作。
巨集病毒就是把惡意程式碼藏在這些自動指令裡，當你開啟檔案並啟用巨集時，病毒就會偷偷執行，可能感染你的電腦或傳播給別人。

🔍 入侵防禦系統（IPS）是甚麼？
IPS 就像「電腦網路的保全警衛」，會主動偵測並阻止可疑或惡意的網路流量或攻擊。
它不只像防火牆阻擋壞流量，還會分析資料包的內容，確認是不是病毒或攻擊，然後攔截它們。

🔍 防火牆或入侵防禦系統（IPS）可以利用簽章檔過濾掉這類惡意流量，是甚麼意思？
簽章檔就像病毒或攻擊的「指紋資料庫」。
防火牆和 IPS 會檢查網路資料是否跟簽章檔中已知的攻擊行為或病毒特徵吻合，如果吻合，就會攔截、不讓它進來，保護電腦和網路安全。

🔍 SMB v1 漏洞（Server Message Block）是甚麼？
SMB 是 Windows 用來分享檔案和印表機的通訊協定。
SMB v1 是這協定的第一版，有個安全漏洞，就是黑客可以利用它入侵電腦。這漏洞就是 WannaCry 等病毒入侵的主要途徑。

🔍 病毒種類簡單解釋：
應用程式型病毒		病毒藏在你平常用的程式裡。你開程式時，病毒也跟著跑，開始壞壞。
開機磁區病毒		病毒藏在電腦開機時用的特別地方（就像電腦的啟動按鈕旁邊），連系統還沒開始工作，病毒就跑出來了。
腳本病毒			病毒寫成一串指令（腳本），藏在系統或網頁裡，只要這串指令被執行，病毒就開始工作。
巨集病毒			病毒躲在像 Word 或 Excel 這些檔案裡的自動指令（巨集）裡，你開啟檔案並允許巨集後，病毒就被啟動了。
=================================================================================
IPS 的全名是：Intrusion Prevention System（入侵防禦系統）

它的功能是：
🔍 偵測駭客攻擊行為（像是惡意流量、病毒程式碼）
🛡️ 並在攻擊成功前主動阻擋或封鎖這些威脅。

簡單來說，它不只是「看見問題」，還會「主動防守」
=================================================================================
🧨 SMB v1 的安全漏洞是什麼？
SMB v1（Server Message Block version 1） 是微軟早期開發的網路檔案分享協定，用來讓電腦之間傳輸檔案、印表機等資源。

🛑 漏洞細節（CVE-2017-0144）
這個漏洞的編號是 CVE-2017-0144，它是一個「遠端執行程式碼漏洞」（Remote Code Execution，簡稱 RCE），存在於：
Windows XP、Windows 7、Windows Server 2003、2008…等許多老版本系統使用 SMBv1 協定 的情況下

⚠️ 這個漏洞的問題是：
只要有開啟 SMB v1 功能，駭客就能遠端發送特殊封包（packet），不需要密碼，也不需要使用者點擊任何東西，就能直接在你的電腦中執行任意程式。

🕵️‍♂️ 黑客是怎麼利用這個漏洞入侵的？
這場攻擊最出名的就是 WannaCry 勒索蠕蟲（WannaCrypt）。

🛠️ 他們使用了 NSA（美國國安局）外流的工具：
一個叫 EternalBlue（永恆之藍） 的攻擊工具
是用來利用 SMBv1漏洞（CVE-2017-0144） 的一個自動化入侵程式

💥 WannaCry 的入侵流程如下：
1️.掃描		感染電腦會掃描整個區網，看哪台電腦還開著 SMB v1
2️.發送攻擊	透過 EternalBlue 發送惡意封包，利用漏洞讓遠端電腦執行程式
3️.安裝後門	安裝名為 DoublePulsar 的後門程式
4️.下載病毒	從網路上下載 WannaCry 病毒主程式
5️.加密勒索	開始加密電腦裡的資料，並要求支付比特幣贖金
6️.傳播		再去感染下一台電腦，快速擴散全球
=================================================================================
🔐 「永恆之藍（EternalBlue）」 是一個由美國國安局（NSA）開發的駭客攻擊工具，主要用來針對 Windows 系統的一個漏洞（SMB v1 漏洞）進行遠端入侵與控制。
📌 它的用途簡單來說就是：
在不需要密碼、使用者不點擊任何東西的情況下，遠端控制一台 Windows 電腦。

🔍 DoublePulsar 是什麼？
Pulsar(脈衝星)
DoublePulsar 是一個在 Windows 系統上的內核級後門（kernel-level backdoor），一旦成功植入目標電腦，攻擊者就可以：
執行任意的惡意程式碼
注入 DLL 檔案（通常是 payload）
繼續進行遠端控制或橫向移動
安裝其他惡意工具

📌 Remote Access Trojan (RAT) 是什麼？
RAT（Remote Access Trojan）遠端存取木馬，是一種惡意程式，讓攻擊者可以秘密控制受害者的電腦或裝置，就像攻擊者坐在該電腦前操作一樣。

🔍 RAT 的主要特性包括：
遠端控制：攻擊者可操控滑鼠、鍵盤、桌面、文件、攝影機、麥克風等。
資料竊取：例如密碼、瀏覽歷史、帳號、金流資料等。
監控能力：錄音、截圖、錄影、記錄鍵盤輸入（keylogger）。
隱蔽性：通常會偽裝成合法檔案，如遊戲破解、免費軟體、影片等。
自動開機啟動：一旦被感染，重開機後仍能運行。
後門功能：讓攻擊者持續存取裝置，甚至更新木馬。

💣 常見用途：
間諜活動（如政府、企業間諜）
勒索攻擊前置準備
安裝其他惡意軟體（如挖礦程式、蠕蟲、病毒）
青少年或業餘黑客用於惡作劇、個人監控（極具風險且違法）

🐍 DarkComet RAT 是什麼？ (DarkComet Remote Access Trojan) 暗彗星
DarkComet 是一款非常知名的 RAT 工具，由法國開發者 "DarkCoderSc" 原本在 2008 年左右開發出來，
初衷是做為合法遠端管理工具（Remote Administration Tool），但後來大量被用於惡意用途。

⚠️ DarkComet 的功能特色：
控制目標的桌面畫面（Remote Desktop）
操控攝影機與麥克風
發送假訊息或對話視窗
瀏覽與下載檔案
鎖定或重新啟動系統
Keylogger（鍵盤側錄）
建立「持久性」，避免被關掉或刪除

🌍 為什麼它知名？
免費且容易取得（儘管開發者已停止釋出）
使用者介面友好
曾被恐怖組織或間諜活動使用（例如敘利亞內戰期間，有用 DarkComet 監控異議份子）

🚫 開發者聲明：
DarkComet 的原始開發者 在 2012 年停止開發並公開反對其惡意用途，呼籲大家不要再使用它來從事非法活動。

🔐 對抗 RAT 的建議：
✅ 安裝並定期更新防毒軟體、反間諜軟體。
⛔ 不要下載來源不明的檔案或點擊可疑連結。
🔍 使用防火牆監控異常連線。
🛑 停用不必要的遠端桌面功能與執行權限。
👁️ 定期檢查系統啟動項目與背景程序。

🕳️ 一、什麼是「後門（Backdoor）」？
後門（Backdoor）是指繞過正常認證或安全機制，秘密地進入系統的方法。
它可以是程式碼、帳號、漏洞、或系統設定。
目的：讓攻擊者能持續控制或存取系統，而不被發現。
